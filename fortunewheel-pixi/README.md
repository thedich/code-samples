# Fortune Wheel by Pixi.js 
`en:` Code sample build [Pixi.js](http://www.pixijs.com/) & [Webpack](https://webpack.js.org/) & ES6\
`ru:` Пример сборки [Pixi.js](http://www.pixijs.com/) & [Webpack](https://webpack.js.org/) & ES6

## Install & Run

```sh
cd fortunewheel-pixi 
npm install
npm run dev     
npm run server
lookup http://localhost:8080/
```
